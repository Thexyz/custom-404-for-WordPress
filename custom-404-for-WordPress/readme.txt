Easily set a custom 404 page URL through wp-admin. This plugin is brought to you by Thexyz.

== Description ==

Custom 404 Page by Thexyz allows you to easily set a custom 404 error page with just a few clicks. Navigate to your wp-admin settings to set a URL of your choice. You can use this with custom domains or sub-domains, such as sitename.thexyz.com.

This plugin offers:

* Simple setup from wp-admin settings.
* Compatibility with custom domains and sub-domains.
* 404 redirect to a URL of your choice.
* Author and support provided by Thexyz.

For more information, visit [Thexyz](https://www.thexyz.com).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-404-by-thexyz` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Custom 404 Page screen to configure the plugin.
